1.<!DOCTYPE>  - Defines the document type
2.<html> -	Defines the root of an HTML document
3.<head> - Contains information for the document
4.<title> - Defines a title for the document
5.<link> -	Defines to link to style sheets
6.<body> - Defines the document's body
7.<header> - Defines a header for a document or section
8.<nav> - Defines navigation links
9.<ul> - Defines an unordered list
10.<li> - Defines a list item
11.<div> - Defines a section in a document
12.<section> - Defines a section in a document
13.<br> - Defines a single line break
14.<center> - Defines centered text
15.<h1 -h6> - Defines HTML headings
16.<b> - Defines bold text
17.<em> - Defines emphasized text 
18.<p> - Defines a paragraph
19.<strong> - Defines important text
20.<img> - Defines an image
21.<table> - Defines a table
22.<th> - Defines a header cell in a table
23.<td> - Defines a cell in a table
24.<tr> - Defines a row in a table
25.<ol> - Defines an ordered list
26.<a> - Defines a hyperlink
